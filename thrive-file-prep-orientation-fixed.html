Thrive File Prep auto-detect emergency cards

Upload Regpack ZIP files. Each ZIP should contain one child PDF and one child photo. The tool keeps only the required child-file sections, places the child photo, duplicates the emergency card pages, and exports finished PDFs.

Everything runs locally in your browser. ZIPs, PDFs, and photos are not uploaded anywhere by this tool.

Photo orientation fix: portrait and landscape phone photos are corrected from orientation metadata before being placed.

or choose files below. You can select all child ZIP files at once.

This version does not assume the Emergency Contact Card is on the same page every time. It reads each PDF and detects pages containing Emergency Contact Card, so it still works when earlier pages move because parent responses are longer.
