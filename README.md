# Regpack Tools

Static browser tools for processing Regpack CSV exports.

## Files

- `index.html` — landing page
- `tshirt-counter.html` — t-shirt quantity counter
- `first-week-file-list.html` — first-week file-making list

## Privacy

These tools run locally in the user's browser. CSV files are not uploaded anywhere by the tools.

Important: do not commit/upload real registration exports, child information, or parent information to this GitHub repository.
