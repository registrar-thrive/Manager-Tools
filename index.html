Manager Tools

Browser-based tools for processing Regpack CSV exports and Thrive program files.

Files

Thrive File Prep

Upload child ZIP files, keep required sections, fix photo orientation, duplicate emergency cards, and export prepared PDFs.

 Attendance

Weekly Attendance Lists

Upload locked weekly/daily registration exports, see weekly totals, and print alphabetized attendance lists.

 Files

First-Week File List

Upload summer registration exports and list each child under the first week they attend.

 T-Shirts

T-Shirt Counter

Upload Regpack CSV exports and total Youth Small, Medium, and Large shirt quantities.
Privacy note: these tools run in the browser. CSV, ZIP, PDF, and photo files are read locally by the page and are not uploaded to a server. Do not upload real Regpack CSV exports, child photos, PDFs, or ZIP files to GitHub.
